🌐 Web Pages Project

A collection of static HTML pages built with HTML and CSS, including a profile card, feedback form, and registration form.


📁 Project Structure

├── .vscode/          # VS Code workspace settings
├── card.html         # Profile/business card page
├── feedback.html     # User feedback form
├── register.html     # User registration form
└── tom.jpeg          # Image asset used in the project

📄 Pages Overview

🪪 card.html
A profile or business card page that displays user information in a card layout. Uses tom.jpeg as the profile image.

📝 feedback.html
A feedback form page where users can submit their thoughts, ratings, or comments.

📋 register.html
A registration form page for collecting user details such as name, email, and password.

🛠️ Built With

HTML5 — Page structure and content
CSS3 — Styling and layout
